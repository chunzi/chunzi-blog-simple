一个写 Perl 的程序员终于不堪忍受日常繁复的 Javascript 编写工作，想要建立一个和 Perl 的 CPAN 类似的 Javascript 的代码仓库，以便高效率的管理和复用那些看起来不难却很支离破碎的 JS 代码。于是就有了 JSAN。关于 JSAN。 现在 Ajax 流行，各种相关的 JS 库纷纷推出，用法各异。对于自动化表单验证，表格逐行不同背景色渲染，操作 DOM 树等等，都已经有相关的 Javascript 代码片断或者 .js 文件可供使用，不过这个世界看起来还是那么凌乱。 这个 JSAN 让我非常激动，它的构架和 CPAN 极为相像，这也难怪，本来就是借鉴 CPAN 的，况且是 Perl 程序员开发的。更为可贵的是，它是一个就在今年 6 月份才开始的开源项目，到目前为止已经初具规模，其执行力可见一斑。 此外，还有个 Planet Javascript 站点，聚合了相关领域的一些站点的 Feeds，也就是合烧了。
