Ubuntu 下看电视 我喜欢看的也就是纪实频道了。接下来是新闻，吃饭的时候可能会看《开心公寓》或者《老娘舅》。 偶尔看到介绍使用 p2p 的方式看电视，自然小试一把： 
sudo apt-get install libstdc++5 libgtk2.0-dev libasound2-dev checkinstall
wget http://download.sopcast.com/download/sp-sc.tgz
tar zxvf sp-sc.tgz
sudo cp ./sp-sc/sp-sc /usr/local/bin/
wget http://dengpeng.name/upload/gsopcast_0.2.10-1_i386.deb
sudo gdebi gsopcast_0.2.10-1_i386.deb
这个时候就可以打开客户端： 
$ gsopcast
从里面选频道，然后就可以在 mplayer 里面看电视了。
