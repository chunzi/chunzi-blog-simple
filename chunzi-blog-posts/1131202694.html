Text::Greeking 发现一个有趣的的模块 Text::Greeking，用来将一堆文字弄得模糊些。 以前看到邮件列表中，国外的设计人员展示一个 web 页面设计的时候，将文字内容的地方填充了很多我看不懂的文字。当初以为是德文或者其他什么语言，今天看到这个模块，才恍然大悟。正如该模块文档中所说的： This is a module is for quickly generating varying meaningless text from any source to create this illusion of the content in systems. 虽然看起来像一篇什么文章，不过却是毫无意义的文字。看起来好像无实际用处的模块，不过满好玩的，至少弄出来的东西让人家愣上一愣，嘿嘿。
